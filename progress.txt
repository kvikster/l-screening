ПУНКТ 1. Blank-subtraction ПЕРЕД merge паралельних проб   [IN PROGRESS]
────────────────────────────────────────────────────────────────────────────────

Поточний порядок (process_peaks_inner у lib.rs):
  1. coarse_screen → ConfirmedRow на (sample_type, polarity)
  2. розділення blanks / samples
  3. merge_parallel_samples(samples) → final_results (усереднення між sample_1, sample_2 …)
  4. blank_candidates + apply_blank_result для кожного merged peak

Проблема: Status (Artifact / Real Compound) призначається на вже усередненому ряді.
Якщо одна проба чиста, друга має шум на тій же RT — після усереднення це може
маскуватися або, навпаки, гасити реальний сигнал blank-ом, що збігся лише з однією
з проб.

Новий порядок:
  1. coarse_screen
  2. розділення blanks / samples
  3. **Per-sample blank subtraction**: для кожної ConfirmedRow з sample-родини
     обчислюється власний blank match → status, signal_to_blank_ratio,
     blank_area_mean, area_difference, confidence_score.
  4. **merge_parallel_samples(samples)**: при злитті агрегуємо нові поля:
       - blank_area_mean: зважене середнє по replicate_count, тільки серед джерел
         що мали blank match;
       - area_difference: area_mean(merged) − blank_area_mean(merged) — або зважене
         середнє існуючих area_difference (рішення нижче);
       - signal_to_blank_ratio: area_mean(merged) / blank_area_mean(merged); якщо
         жоден source не мав blank match → None;
       - status: переоцінюється з агрегованого S/B vs signal_to_blank_min;
       - confidence_score: перераховується через final_confidence_score
         (replicate_score(merged), has_blank_match, ratio).
     Per-source значення зберігаються у Why.SourceRows для аудиту.

Рішення з трейд-офів:
  - **Area_difference на merged** = area_mean(merged) − blank_area_mean(merged).
    Це консистентно з тим, як рахується S/B ratio (через агрегати, не середнє відношень).
  - **Status агрегованого ряду** виводиться з агрегованого S/B, а не "any Real Compound
    серед джерел". Це уникає ситуації коли одна шумна проба псує висновок.
  - **Якщо лише частина джерел мала blank match** (наприклад: sample_1 збігся з blank,
    sample_2 — ні). Тоді blank_area_mean агрегується тільки по тих джерелах, що мали
    збіг. Has_blank_match = true якщо хоча б одне джерело мало match.
