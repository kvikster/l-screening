# Реліз v0.8.0

`v0.8.0` вводить LabSpaces — singleton workspace для керування blank, surrogate, analysis datasets, analyzers та історією звітів.

## Що змінилося для користувача

- **LabSpaces**: новий workspace для централізованого керування всіма даними:
  - Зберігайте blank, surrogate та analysis datasets в одному місці
  - Керуйте analyzer presets з параметрами скринінгу
  - Переглядайте історію всіх створених звітів
  - Перемикайтеся між dataset-ами без повторного завантаження файлів

- **Persistent storage**: всі LabSpaces дані зберігаються локально в браузері
  - Automatic persistence при кожній зміні
  - Миттєве відновлення стану після перезавантаження
  - Export/import LabSpaces для резервного копіювання

- **Improved workflow**:
  - Швидке перемикання між різними dataset-ами
  - Повторне використання analyzer presets
  - Доступ до історії звітів без повторного обчислення

## Що це означає на практиці

- Більш ефективна робота з великими наборами даних — завантажте один раз, використовуйте багато разів
- Краща організація роботи з різними типами даних (blank, surrogate, analysis)
- Можливість швидко порівнювати результати з різними параметрами скринінгу
- Зменшення часу на повторну обробку даних
