Рекомендовані поля для експорту з Agilent MassHunter

Призначення: ця таблиця допомагає швидко узгодити, які поля варто експортувати з Agilent MassHunter для алгоритму аналізу піків, якості сигналу та виявлення артефактів.

Контекст: Agilent 1260 Infinity II + Agilent Ultivo LC/TQ G6465B + типові LC/TQ workflow у MassHunter.

Що брати в першу чергу
  • Minimum: базова ідентифікація зразка, аналіта і піка.
  • Good: усе для практичного scoring і artifact detection.
  • Ideal: усе для розширеної QA/QC і кращої моделі.
Найцінніші поля саме для artifact detection
