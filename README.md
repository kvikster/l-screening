# LC-MS Screening Assistant

Progressive Web App (PWA) for LC-MS data screening and analysis with a modern SvelteKit app and Python FastAPI server.

## Features
- LC-MS data screening and analysis.
- Modern Svelte 5 app with offline PWA support.
- Python FastAPI server for data processing.
- Deployable to GitHub Pages or any static hosting.
- Service worker for offline functionality.

## Prerequisites

### Frontend
- Bun (for package management and running)
- Rust and Cargo (for building WebAssembly modules)
  - Install wasm-pack: `cargo install wasm-pack`

### Backend
- Python 3.10+
- pip (for installing Python dependencies)
